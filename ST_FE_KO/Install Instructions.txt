
To use IIS or Apache:
1.  Copy source files into IIS, or Apache and run accordingly

To use NodeJS:
1.  Install the lateste version of NodeJS from https://nodejs.org/en/download/
2.  Copy source files from Github to location on the local machine
3.  Using the command prompt or terminal go to the location of where the source files were saved.  Type 'npm install'.  This should install all of the npm modules
4.  Once the npm install is complete, in the command prompt type 'npm start'. This should kick off the application.  
		**NOTE** in the cmd/terminal window  you'll see a console.log that shows the port location for the app.   ie....http://localhost:5500/
